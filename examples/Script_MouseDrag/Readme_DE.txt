--- Script_MouseDrag ---

0. Description
Verhindert die fälschliche Erkennung von Maus-Drag-Gesten im Inventar.
Das Problem tritt nur unter bestimmten Bedingungen auf
(z.B. auf Windows 10 oder wenn Eclipse im Hintergrund läuft).

1. Installation
Der Ordner 'scripts' muss ins Gothic 3 Installationverzeichnis kopiert werden.
Es ist nicht notwendig ein neues Spiel zu beginnen.
