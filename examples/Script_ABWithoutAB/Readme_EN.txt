--- Script_LearnPoints ---

1. Content
Some changes that are normally only available in combination with alternative balancing are now also availabe without it:
- After learning the perk "Masterthief", the hero can commit thefts in cities without being taken to task for them (as long as nobody witnesses the theft).

2. Install
The 'scripts' folder has to be copied into the Gothic 3 install directory.
There is no need to start a new game.

3. Uninstall
Remove the file Script_ABWithoutAB.dll from the 'scripts' folder in the Gothic 3 install directory.
